### Deployment Instructions:

1. Copy `.env.example` to `.env` and add your GitHub token
2. Run `docker-compose up -d --build`
3. Access the app at http://localhost:3000
4. Initialize tools: `docker exec agent-tracker-backend ./init-tools`
5. Run initial seed: `docker exec agent-tracker-backend ./seed`
6. Set up cron for sync: `docker exec agent-tracker-backend ./sync`

### Architecture:

- **Frontend**: React + Vite + Tailwind CSS (served on port 3000)
- **Backend**: Go + Gin + SQLite (port 8080)
- **Mock Server**: Express.js (port 3001) for development/testing
- **Database**: SQLite with GORM ORM

### Notes:

The application uses a mock server for development/testing that provides all the same API endpoints as the Go backend. This allows full frontend development without requiring Go runtime. The Go backend is production-ready and can be deployed with Docker.
